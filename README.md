# twr-wireshark
Wireshark dissector to show data needed to analyse UDP packages send by Ranging Kit from DW.
To get the data an UWB Sniffer is used.

## sample-file
Sample file with two cycles, can be used to look at the result. This is real but anonymized data.

## Wireshark
Tested with 3.6 on Linux and 4.6 on Windows. You have to put the file in the plugin folder of your installation and activate the protocol.
